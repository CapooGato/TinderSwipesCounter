#### What does it do
Counts how many likes and passes have you done on your Tinder account.

#### Requirements
- data.json file which you can obatain by sending a request to Tinder, they send it in usually few days.
- python version 3.* as the the program uses only json library and built-in dictionary operations.

#### Output
Swipes in total: `<number>`  
Likes: `<number>`\
Passes: `<number>`

Percentages: 
Likes: `<number>`%\
Passes: `<number>`%

#### Running the program
- insert data.json file in the same directory as 'swipes\_couner\_tinder.py' file
- run the script
